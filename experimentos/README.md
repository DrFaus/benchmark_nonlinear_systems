* factor 1 -> CSTR: 0 moderadamente no lineal, 1 altamente no lineal
* factor 2 -> FIS Type: 0 Mamdani, 1 Tsukamoto
* factor 3 -> Parametrización: 0 Discreto, 1 continuo
* factor 4 -> 0 incremental-form, 1 level-form

Nombre del driver: driver_<factor 1><factor 2><factor 3><factor 4>.exe

Todos los drivers están hechos para correr en Windows. 

Al ejecutar el .exe comienza el proceso de optimización, lo cuál dará como resultado un csv con los resultados de la optimización.
